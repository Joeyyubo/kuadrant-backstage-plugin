{% raw %}##{% endraw %} Deployment

### GitHub Pages (Automatic)

This portal is automatically configured to deploy to GitHub Pages using GitHub Actions. 

**No manual configuration needed!** The workflow will:
- Automatically deploy when you push to the `main` branch
- Enable GitHub Pages in your repository settings
- Make your portal available at `https://<username>.github.io/<repo-name>/`

The GitHub Pages URL is already configured in the `catalog-info.yaml` file.

### Other Hosting Options

- Netlify
- Vercel
- AWS S3 + CloudFront
- Azure Static Web Apps
